# MediPlan Komponentenbibliothek

## Verfügbare Basisklassen

### Buttons

```html
<button class="mds-button mds-button--primary">Speichern</button>
<button class="mds-button mds-button--secondary">Abbrechen</button>
<button class="mds-button mds-button--danger">Löschen</button>
```

### Karten

```html
<article class="mds-card">...</article>
```

### Formulare

```html
<div class="mds-field">
  <label>Name</label>
  <input class="mds-input">
</div>
```

### Status

```html
<span class="mds-badge mds-badge--success">Bestand OK</span>
```

### Layout

- `mds-stack`
- `mds-cluster`
- `mds-enter`
