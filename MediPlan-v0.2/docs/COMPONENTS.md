# Komponenten

## App Header
Enthält App-Name, Begrüßung, Datum und Design-Umschalter.

## Hero Card
Große Übersichtskarte auf dem Dashboard.

Inhalt:
- Einnahmen offen
- Termine
- Bestand
- Hinweise

## Status Pill
Kleine Statuskarte mit Icon, Wert und Beschreibung.

## Intake Card
Karte für eine offene Medikamenteneinnahme.

Inhalt:
- Medikamentenbild oder Icon
- Name
- Stärke
- Uhrzeit
- Menge
- Button „Jetzt einnehmen“

## Mini Card
Kleine Karte für ergänzende Informationen wie Termin oder Notfall.

## Bottom Navigation
Feste Navigation am unteren Bildschirmrand.

Tabs:
- Start
- Einnahmen
- Schrank
- Kalender
- Mehr
